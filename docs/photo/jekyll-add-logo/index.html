用照片制作svg

这次只描了轮廓

  • 用钢笔工具和选择工具画路径:
    • 在最凸，最凹的地方加锚点
    • 添加锚点时拉拽得到切线
    • 按command 单独调整锚点和切线的端点
    • option调整切线
    • 用直接选择工具(白色箭头)调整锚点位置
    • 用路径选择工具(黑色箭头)复制路径
    • 用颜色填充功能实时查看路径闭包
  • 导出:
    • 选中填充图层复制svg

设置网页的favicon

  • 在对应html设置icon: <link rel="shortcut icon" type="image/x-icon" href="/personal_homepage/docs/images/logo.ico">
    • 我的路径是head.html->head-custom.html
